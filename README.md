## Deployment Instructions

### GitHub Pages Setup

1. **Repository Setup**
   - Fork or clone this repository
   - Ensure all files are in the root directory

2. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Source: Deploy from a branch
   - Branch: main (or master)
   - Folder: / (root)
   - Click Save

3. **Domain Configuration**
   - In Pages settings, add custom domain: `crystalbuildingmaintenance.com`
   - Enable "Enforce HTTPS"

4. **DNS Configuration (GoDaddy)**
   ```
   Type: CNAME
   Name: www
   Value: yourusername.github.io
   
   Type: A
   Name: @
   Value: 185.199.108.153
   Value: 185.199.109.153
   Value: 185.199.110.153
   Value: 185.199.111.153
   ```

### File Updates Needed

1. **Logo Integration**
   - Replace `logo.jpg` placeholder with actual company logo
   - Recommended size: 120x120px, PNG format with transparent background
